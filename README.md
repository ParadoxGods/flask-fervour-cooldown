# Flask Fervour Cooldown

Shows the Leagues 6 Flask of Fervour cooldown in game ticks and highlights the flask when it is ready.

Features:
- Reads the game's Flask of Fervour cooldown clock directly
- Falls back to click, chat message, and damage tracking if the cooldown clock is unavailable
- Draws tick text, cooldown shading, and a progress bar over the flask in your inventory or equipment
- Highlights the flask when the cooldown reaches zero
- Sends a chat message when the flask is ready showing the seconds and ticks saved that cooldown

Notes:
- This plugin is only for the Leagues 6 Flask of Fervour.
- It only uses item overlays and does not add a movable panel or sidebar panel.
